Besides 5 menu types YJSG comes with additional 3 Bootstrap menus. To activate Bootstrap menus, go to your Menu Module manager, tab name Yjsg Menu Options and from parameter Select menu type, select either Bootstrap nav pills, Bootstrap navbar or Bootstrap navbar inverse.


[yjsgnote color="blue" close="no" title="Bootstrap menu info" border="default" radius="radiusb5" icon="fa fa-info-circle fa-lg"]

Mega menu is not functional when you use Bootstrap menu styles and Bootstrap menu type will show maximum of 3 menu levels. Note that Bootstrap menu requires either Bootstrap2 or Bootstrap3 to be activated in template manager.

[/yjsgnote]
